# three-in-a-row-socket-multi-threaded
<p>This program implements a 'three in a row game'. It uses C socket and multi-threading. </p>
<ul>
  <li>
    <h4>Compilation</h4>
    <ul>
      <li>g++ version > 4.7, recommended... </li>
      <li>Linux OS - 'g++ -std=c++0x three-in-a-row.cpp -o <b>three-in-a-row</b> -pthread'</li>
    </ul>
  </li>
  <li>
    <h4>Execution</h4>
    <ul>
      <li>In the executable(<b>three-in-a-row</b> binary file) folder:</li>
      <li>Linux OS  - '<b>.</b>/three-in-a-row'. Server will listen on local host, port 12000.</li>
    </ul>
  </li>
  <li>
    <h4>Server Access</h4>
    <ul>
      <li>Linux OS  - 'telnet 127.0.0.1 12000'</li>
    </ul>
  </li>
</ul>
